== Description ==

cart page cart box quantity calculation without update button extention for woocommerce


= Using the Plugin =

* Download the plugin, install and active.

* Now go to <strong>shop</strong> > <strong>page</strong> > <strong>cart</strong>, location URL would be like below -

`
http://yourwebsite.domain/cart/
`

* Now you will see few default setup.


== Installation ==

You may install the plugin using one of the three following methods:

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Then activate the plugin.
